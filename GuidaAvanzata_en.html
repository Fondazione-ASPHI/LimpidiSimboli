1.0 Introduction to Limpidi Simboli: A Strategic Tool for AAC

In the field of Augmentative and Alternative Communication, having flexible, powerful, and accessible tools is essential to effectively support learning and autonomy. Limpidi Simboli, the web app developed by Fondazione ASPHI Onlus, integrates the tradition of symbol sets with the potential of artificial intelligence and the web.

  • Completely free: The absence of costs for all basic functionalities makes the tool immediately available without requiring initial investment.
  • Customizable: The ability to upload, generate, search for, and save custom symbols allows for the creation of materials that reflect the specific life experiences of individuals (photos of family, familiar places, personal items), increasing engagement and communicative effectiveness.
  • Intelligent: Unlike older tools based on simple word-to-symbol matching, Limpidi Simboli integrates advanced grammatical recognition. It can analyze tenses, gender, and number, providing a richer and more accurate visual representation of the sentence.

To unlock the full potential of this tool, it is essential to start with the correct initial setup. This first step is the foundation for an efficient workflow and for ensuring that every customization is saved permanently and securely.

2.0 Initial Setup: Preparing the Work Environment

Quick Setup Guide

Follow these steps to prepare your work environment optimally.

  • Step 1: First Launch and Accessibility

    Launch the application in a modern web browser. For a full and unrestricted experience, the use of Google Chrome, Microsoft Edge, or Opera is recommended. Currently, browsers like Firefox and Safari do not support the File System API, a crucial technology for saving images to disk, limiting some advanced customization features.

  • Step 2: Configure the Image Folder (Crucial)

    This is the most important step to ensure the persistence of your work. Click the "üìÇ Select Image Folder" button. You will be asked to create or select a folder on your computer (e.g., in Documents/AAC_Materials/Limpidi_Simboli).
    Why it's crucial: Without this link, every custom image (uploaded from your PC or generated with AI) is saved temporarily in the browser's memory (localStorage), which has space limits and can be cleared. By configuring the folder, images are saved permanently to your local disk, avoiding browser memory saturation and ensuring your custom symbols are always available.

  • Step 3: Set up Text-to-Speech

    For clear and natural audio output, go to settings (‚öôÔ∏è Settings) and, in the text-to-speech dropdown menu, select the "Google" voice for your language (e.g., "Google UK English Male"). This voice, generally integrated into the browser, offers significantly higher quality than default system voices.

  • Step 4: Configure APIs (Optional but Powerful)

    This step is optional but unlocks the application's most advanced and intelligent features. In the "‚öôÔ∏è Settings" panel, you can enter the following API keys:

    • OpenAI API Key: Enables enhanced grammatical analysis via GPT (for recognizing implicit pronouns and greater accuracy) and custom image generation with DALL-E.
    • OpenSymbols Token: Allows access to alternative and complementary symbol libraries to ARASAAC, such as TAWASOL or Bliss, enriching your visual toolbox.
    • Google API Key and CX: Unlocks image search directly from Google, with the benefit of active SafeSearch and a generous daily limit of free searches.

Privacy Guaranteed: All API keys and settings are saved exclusively locally in your browser. No sensitive information is sent or stored on external servers.

3.0 Essential Functionality: Daily Free Operation

3.1 Text-to-ARASAAC-Symbol Translation

  • What it does: Converts a written sentence into a sequence of pictograms from the ARASAAC set.
  • How to use: Enter the desired text in the field, select the language (Italian, Spanish, or English), and press the "üó£Ô∏è Translate" button. You can activate a filter to automatically hide articles and prepositions, resulting in a cleaner visual strip focused on key concepts.

3.2. Basic Grammatical Analysis

  • What it does: Adds visual badges above symbols to indicate essential grammatical information.
  • How to use: The function is automatic. Once the sentence is translated, badges appear on verbs and nouns, indicating: ‚Äπ Past, ‚ñ∂Ô∏è Present, ‚Ä∫ Future, 1Ô∏è‚É£ Singular, + Plural.

3.3 Management and Customization of Individual Symbols

These direct actions on each symbol allow for granular control over the final visual representation.

  • See Alternatives: A word often has multiple symbolic representations. By clicking repeatedly on a symbol, you can browse all available alternatives in the ARASAAC database and choose the one most relevant to the context.
  • Text-Only Mode ("abc"): This function hides the image and displays only the written word in large print. It is a valuable tool for facilitating the transition from symbol to written word for users developing literacy skills.
  • Upload a Personal Image ("+"): By clicking the + button, you can upload an image or photo directly from your computer (e.g., a photo of your dog, your school). If the local folder has been configured, the image is saved permanently and associated with that word for future use.

  • What it does: Allows you to merge two or more symbols into a single composite pictogram.
  • How to use: While holding down the CTRL key (or Cmd on Mac), select the symbols to merge. The "üîó Merge" button will appear. Clicking it will create a new symbol representing the complex concept.
  • This is an essential function for representing abstract concepts, complex actions, or compound nouns that do not have a dedicated symbol (e.g., "camera" + "bed" ‚Üí "bedroom"; "go" + "away" ‚Üí "go away").

3.2.5. Using Text-to-Speech ("üîä Listen")

  • What it does: Reads the entered sentence aloud.
  • How to use: By clicking the "üîä Listen" button, the browser's integrated text-to-speech is activated.

4.2.2. Custom Symbol Generation with DALL-E (OpenAI)

This is one of the most powerful features for customization, allowing you to create an image for any concept.

5.0 Managing Your Custom Symbol Library

Orderly and conscious management of custom symbols is strategically important. Over time, your library becomes a valuable professional asset that reflects your working method and the specific needs of your clients.

By accessing the "Custom Symbols" panel within the settings (‚öôÔ∏è Settings), you will have the following functionalities available:

  1. View and Control: You can access a complete list of all the symbols you have saved. For each word, the interface shows a preview of the associated images.
  2. Selective Deletion: The system offers three levels of control:
    • Delete a single symbol: Clicking the "x" next to an image removes that specific association.
    • Delete all symbols for a word: Removes all custom images associated with a term, restoring the default ARASAAC behavior for that word.
    • Total deletion: Drastic option that, upon confirmation, deletes the entire library of custom symbols.
  3. Backup and Sharing (Export): The "üíæ Export" function generates a JSON file containing all associations. This file, combined with your local image folder, constitutes a complete backup of your library.

Important Note: The Symbol Hierarchy
Always remember this fundamental rule: the symbols you have customized (uploaded, AI-generated, chosen from the web, or selected as favorites) will always take precedence over those proposed by default by ARASAAC.

7.0 Frequently Asked Questions (FAQ)

  • How much does it cost to use Limpidi Simboli?

    The application and all its basic features are completely free. Costs are associated exclusively with the optional use of external API services:

    • OpenAI (GPT & DALL-E): Pay-as-you-go costs (text analysis is very cheap, image generation is $0.04/image).
    • Google Search: Offers 100 free image searches per day.
    • OpenSymbols: Access is free, only requires registration on their site for a token.
  • Is my data secure?

    Yes. Privacy is a priority. All data (API keys, custom images, preferences) are saved exclusively on the user's computer (in browser storage or the designated local folder) and are never sent to external servers.
    Disclaimer: When using OpenAI APIs, the sentences sent for analysis or image generation are subject to their privacy policy.

  • Can I use the application in a school context?

    Absolutely. It is an ideal tool for inclusive education.

  • What are the limitations on commercial use?

    Clinical, therapeutic, and educational use to create materials for one's own patients or students is fully permitted. However, it is important to note that ARASAAC symbols are released under a CC BY-NC-SA license. This means that materials created using these symbols cannot be sold.

  • Which browsers are fully supported?

    To ensure access to all features, including file management on disk, the use of Google Chrome and Microsoft Edge is recommended. Browsers like Firefox and Safari currently do not support the File System API.

B. Glossary of Technical Terms

  • AAC (Augmentative and Alternative Communication): A set of strategies, tools, and techniques used to support communication for people with complex communication needs.
  • ARASAAC: A vast database of free pictographic symbols, created by the Government of Aragon (Spain).
  • API (Application Programming Interface): A computing "bridge" that allows two different pieces of software to communicate.
  • localStorage: A small storage area within the web browser, used to save data (like settings or symbols) directly on the user's computer. It has space limits.
  • File System API: An advanced web technology that allows a web application, with user permission, to read and write files directly in a specific folder on the computer.
  • GPT (Generative Pre-trained Transformer): An AI model developed by OpenAI, specializing in understanding and generating natural language.
  • DALL-E: An AI model by OpenAI capable of creating realistic and artistic images from a text description.
  • Token: The unit of measurement used by OpenAI to calculate the consumption and cost of its APIs. A sentence is composed of multiple tokens.
